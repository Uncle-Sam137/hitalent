# Chats API (FastAPI)

Тестовое задание: API для чатов и сообщений.

Реализован REST API для создания чатов, отправки сообщений и получения истории сообщений.

---

## Стек технологий

- Python 3.12
- FastAPI
- PostgreSQL
- SQLAlchemy (async)
- Alembic (миграции)
- Docker / Docker Compose
- pytest

---

## Запуск проекта

### Требования
- Docker
- Docker Compose

### Запуск

```bash
docker-compose up --build
```

### После запуска API будет доступно по адресам:

API: http://localhost:8000

Swagger UI: http://localhost:8000/docs
